The cards below showcase some programming projects I've made. Click on them to see their respective repository. You can also go to my Github directly and browse from there.

GroupMe Bot (Python)

I made this bot to help coordinate with coworkers and manage large group chats. This bot can mention everyone in a chat with "@everyone", automatically greet new users who join a chat, and much more! This program is meant to run on the cloud so that the bot is always active.

Algorithm Visualizer (C++)

This executable allows you to see how the A* algorithm works by animating it. You can draw a maze or let the program generate one for you. After that just click start and you'll see the algorithm in action. (Requires the SFML library if you want to compile it yourself.)
