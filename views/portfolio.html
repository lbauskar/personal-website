GroupMe Bot (Python)

I made this bot to help coordinate with coworkers and manage large group chats. This bot can mention everyone in a chat with "@everyone", automatically greet new users who join a chat, and much more! This program is meant to run on the cloud so that the bot is always active.

Kanban (JavaScript)

This extension for Visual Studio Code implements a Kanban Board for every workspace.

Algorithm Visualizer (Java)

This executable allows you to see how pathfinding algorithms work by animating them. You can draw a maze or let the program generate one for you. After that just click start and you'll see the algorithm in action.
